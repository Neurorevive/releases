# Bionic Link — Releases

Official Android releases for the Bionic Link app by Neurorevive.

## Download

**[→ Download latest APK](https://github.com/Neurorevive/releases/releases/latest/download/app-debug.apk)**

## Installation

1. Download the APK above
2. Open the downloaded file on your Android device
3. If prompted, enable **Install from unknown sources** in your settings
4. Follow the on-screen instructions to install

## Requirements

- Android 8.0 (Oreo) or later
- Compatible Neurorevive device

Contact Neurorevive for help with installation or the app.
